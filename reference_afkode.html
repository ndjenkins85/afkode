afkode.utils.split_transcription_on(transcription, words, strategy='after')ÔÉÅ

Uses regex matching based on case-insensitive, matched word boundaries to split text before or after.

  • It matches the exact word you‚Äôre looking for, bounded by non-word characters or the beginning/end of the string.

  • It splits the transcription at these word boundaries.

  • If strategy is ‚Äúafter‚Äù, it takes everything after the last occurrence of the word.

  • If strategy is ‚Äúbefore‚Äù, it takes everything before the first occurrence of the word.

  • If strategy is ‚Äúdetect‚Äù, it will remove the word; this can be tested as a length difference

  • If the word is part of a larger word it doesn‚Äôt match.

  • Case insensitive

Parameters:

  • transcription (str) ‚Äì raw transcription text

  • words (Union[str, List[str]]) ‚Äì single or list of keywords to search and split on

  • strategy (str) ‚Äì whether to return text before or after the word

Raises:

ValueError ‚Äì if an invalid strategy selected

Return type:

str

Returns:

Cleaned transcription text that focuses on info between keywords
